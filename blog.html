Important Faq

What is Document Object Model(DOM)?

The DOM (Document Object Model) represents the structure of a webpage, enabling dynamic content manipulation.

How do you select an element from DOM?

To select an element from the DOM we use getElementById, getElementsByClassName, querySelector, or querySelectorAll to select DOM elements.

What is event deligation in context of DOM, and why is it useful?

Event delegation allows parent elements to handle events for dynamically added or multiple child elements efficiently.

How do you manipulate an events attribute and styles using dom?

We use setAttribute() for attributes, addEventListener() for events, and style for CSS manipulation.
